자주 묻는 질문

데이터 입력한 금액이 다른 기기에서 안 보여요

1억 계산기는 모든 데이터를 기기 내부에만 저장합니다. iCloud 동기화나 계정 로그인을 사용하지 않으므로 기기 간 자동 공유는 되지 않습니다. iCloud 기기 백업이 켜져 있다면 백업/복원 시 함께 이동합니다.

위젯 위젯이 변경한 값을 즉시 반영하지 않아요

설정 저장 또는 입금 확정 직후에는 자동으로 업데이트됩니다. 만약 반영이 늦으면 홈 화면에서 위젯을 길게 눌러 "위젯 편집"을 한 번 열었다 닫으면 새로고침됩니다.

계산 "달성 불가" 표시가 떠요

월 저축 계획이 0원이고 연평균 수익률도 0%면, 현재 보유 금액으로 목표에 도달할 방법이 없어 "달성 불가"가 표시됩니다. 설정에서 월 저축 계획을 입력해주세요.

초기화 모든 데이터를 지우고 싶어요

설정 화면 맨 아래의 "모든 데이터 초기화" 버튼을 누르면 즉시 모든 입력값이 삭제됩니다. 되돌릴 수 없으니 주의하세요.
