1억 계산기 — 개인정보처리방침

최종 업데이트: 2026년 4월 24일

1억 계산기는 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다.

모든 데이터는 사용자의 iPhone/iPad 기기 내에만 저장되며, 앱 개발자를 포함한 어떠한 제3자도 이에 접근할 수 없습니다.

1. 수집하는 정보

앱은 다음 정보를 기기 내부에만 저장합니다:

  • 사용자가 직접 입력한 목표 금액, 현재 보유 금액, 월 저축 계획, 월급일, 연평균 수익률
  • 월별 실제 입금 금액 기록
  • 앱 사용 환경 설정 (예: 온보딩 완료 여부)

이 정보는 외부 서버, 클라우드, 분석 서비스로 전송되지 않습니다.

2. 정보 사용 목적

저장된 정보는 오직 앱의 핵심 기능(목표 달성 예정일 계산, 진행률 표시, 위젯 업데이트, 동기부여 메시지 생성) 수행 용도로만 사용됩니다.

3. 데이터 저장 위치

  • 모든 데이터는 iOS UserDefaults (App Group) 영역에 저장됩니다.
  • iCloud 기기 백업이 활성화된 경우, iOS 시스템에 의해 자동으로 암호화된 상태로 백업됩니다. 이 백업은 Apple이 관리하며, 앱 개발자는 접근할 수 없습니다.

4. 제3자 제공

앱은 수집한 정보를 어떠한 제3자에게도 제공하지 않습니다. 앱 내에 광고, 분석 도구, 추적 SDK가 포함되어 있지 않습니다.

5. 사용자 권리

  • 앱 내 설정 → 모든 데이터 초기화 메뉴를 통해 언제든지 저장된 모든 정보를 즉시 삭제할 수 있습니다.
  • 앱을 기기에서 삭제하면 모든 데이터가 함께 삭제됩니다.

6. 아동 개인정보

본 앱은 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다.

7. 정책 변경

본 방침이 변경될 경우, 앱스토어 업데이트 노트와 이 페이지를 통해 고지합니다.
