## 앱 설명 (Description)

### 첫 3줄 (스크롤 전 노출)

```
월급 200만원, 1억까지 몇 개월?
목표 금액과 월 저축액만 입력하면 달성 예정일을 바로 알려드립니다.
홈 위젯으로 매일 남은 기간을 확인하고, 저축 동기를 유지하세요.
```

### 전체 본문

```
"나는 언제 1억을 모을 수 있을까?"

막연했던 목표를 정확한 날짜로 바꿔드립니다.
목표 금액과 월 저축액, 월급일만 입력하면 복잡한 계산 없이 D-Day가 계산됩니다.

▶ 주요 기능
• 남은 개월 카운트다운: 1억, 3억, 내 맞춤 금액까지 자동 계산
• 진행률과 저축 여정: 시작일, 총 저축액, 다음 이정표 한눈에
• 월 입금 드래그 조정: 실제 입금액이 달라져도 손가락 하나로 반영
• 홈 화면 위젯: Small / Medium 두 크기, 진행률 링과 응원 메시지
• 만약에 시뮬레이션: 월 저축을 조정하면 몇 개월 단축되는지 실시간 확인
• 연평균 수익률 반영: 주식·예적금 복리 계산 포함

▶ 프라이버시 우선
• 모든 데이터는 기기 내부에만 저장
• 광고, 추적, 외부 서버 전송 없음
• 회원가입도 로그인도 필요 없음

▶ 이런 분께
• 1억, 목돈 모으기 목표가 있는 직장인
• 적금/주식 복리 계산을 간편하게 원하는 분
• 가계부보다 목표 하나에 집중하고 싶은 분
• 위젯으로 매일 동기부여를 받고 싶은 분

목표가 있으면 저축은 다릅니다. 오늘 시작하세요.
```
